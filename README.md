# LoveChatbot
<br>
情感諮詢聊天機器人<br>
抓PTT男女版資料訓練doc2vec <br>
找出和輸入問題最接近的K組候選問題<br>
再用BM25比對出K個問題中和輸入問題最相似的問題<br>
然後輸出該問題之PTT回文<br>
